# タッチ操作テスト

設定画面から起動できる「タッチ操作テスト」画面で、注入された操作が端末側で一般的なイベントとして検知されているかを確認できます。

## 起動方法

- 設定画面 → 「動作確認」→「タッチ操作テストを開く」

## 確認できるもの

- タップ
- ダブルタップ
- ロングタップ
- スワイプ（上下左右）

画面上でタッチイベントが発生すると「最終イベント」が更新されます。最初に「タッチ開始」が表示され、その後にジェスチャ判定（タップ/スワイプ等）が表示されます。

## 「最終イベント」が更新されないとき

- アクセシビリティサービスが有効か
- アプリが前面に表示されているか
- 端末側の制限（特定画面/システムUIへの注入ブロック等）がないか

を確認してください。
