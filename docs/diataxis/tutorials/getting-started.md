# はじめに（Getting Started）

ftvrcm をインストールして、Fire TV Stick でタッチ操作を使えるようにするまでの手順を説明します。

## 前提条件

- Fire TV Stick（minSdk 28 / Android 9.0 以上）
- APKをサイドロードするための ADB 環境（PC）

## 1. APK のインストール

### デバッグ APK（推奨：開発・評価用）

```bash
# Fire TV で ADB デバッグを有効化しておく（設定 → マイ Fire TV → 開発者オプション → ADB デバッグ）
adb connect <FIRE_TV_IP>:5555
adb install -r app/build/outputs/apk/debug/app-debug.apk
```

APK のビルド方法は [APK をビルドしてインストールする](../how-to/build-and-install.md) を参照してください。

## 2. AccessibilityService の有効化

本アプリはアクセシビリティサービスとして動作します。インストール後に有効化が必要です。

1. Fire TV の **設定** → **ユーザーコントロール** → **アクセシビリティ**（または「ダウンロードしたサービス」）を開く
2. `ftvrcm` を選択してトグルを **ON** にする
3. アプリの設定画面を開き、「必要状態：アクセシビリティサービス」が **ON** になっていることを確認する

> ADB からも有効化できます。詳細は [AccessibilityService を有効化する](../how-to/enable-accessibility-service.md) を参照してください。

## 3. 基本操作

### 操作モードの切り替え

デフォルトでは **MENU キー長押し** で「通常操作」と「タッチ操作」を切り替えます。

- **通常操作**：リモコンが通常どおり動作（十字キーでフォーカス移動など）
- **タッチ操作**：画面上のカーソルを動かしてタップ・スクロール・スワイプを行う

### タッチ操作モードでの操作

| 操作 | デフォルトキー |
|------|-------------|
| カーソル移動 | 十字キー（上/下/左/右） |
| タップ / ロングタップ | 決定ボタン（短押し / 長押し） |
| 上下スクロール | CH+ / CH- |
| 左右スクロール | 巻き戻し（REW）/ 早送り（FF） |
| カーソル/方向キー切り替え | `mouse_key_cursor_dpad_toggle` に割り当てたキー |

### 設定画面の開き方

Fire TV のホーム画面または「インストール済みアプリ」から **ftvrcm** を起動すると設定画面が開きます。

## 4. タッチ操作を確認する

設定画面 → 「動作確認」→「タッチ操作テストを開く」から、タップ・スクロール・スワイプ・ズームが正しく動作するか確認できます。

詳細は [タッチ操作テストの使い方](../how-to/touch-test.md) を参照してください。
